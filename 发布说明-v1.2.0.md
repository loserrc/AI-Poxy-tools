## 🎯 v1.2.0 新增功能

### 1. SSL 证书验证修复 ✅
- **问题**：连接上游服务器时出现 `certificate verify failed` 错误
- **解决**：集成 certifi CA 证书包，自动验证公共 SSL 证书
- **影响**：无需手动配置，开箱即用

### 2. 端口转发管理 ✅
- **位置**：系统操作区域第三行
- **功能**：
  - 🔄 设置端口转发 (443→8443)
  - ❌ 取消端口转发
  - 👁️ 查看转发规则
- **要求**：需要管理员权限

### 3. 版本更新检测 ✅
- **位置**：帮助菜单 → 检查更新
- **功能**：
  - 自动检测最新版本
  - 显示更新日志
  - 提供下载链接
- **配置**：需要设置 `UPDATE_CHECK_URL`

### 4. Bug 修复
- ✅ Qt 初始化顺序问题
- ✅ 首选项对话框 AttributeError
- ✅ 打包配置优化

## 🚀 快速开始

### 方式 1：单文件 EXE（推荐）

1. **运行程序**：
   ```bash
   dist\Trae-Poxy-UI.exe
   ```

2. **配置更新服务器**（可选）：
   - 在 EXE 同目录创建 `.env` 文件
   - 添加配置：
     ```env
     APP_VERSION=v1.2.0
     UPDATE_CHECK_URL=https://newapi.loserrc.com/trae-poxy/latest.json
     ```

### 方式 2：文件夹版本

1. **运行程序**：
   ```bash
   dist\Trae-Poxy-UI\Trae-Poxy-UI.exe
   ```

2. **配置更新服务器**：
   - 编辑 `dist\Trae-Poxy-UI\_internal\.env`
   - 修改 `UPDATE_CHECK_URL` 为你的服务器地址

## 📋 使用指南

### 基础配置流程

1. **生成证书**：点击"生成证书与配置"
2. **安装 CA**：点击"安装 Root CA"（需管理员）
3. **写入 hosts**：点击"写入 hosts"（需管理员）
4. **设置转发**：点击"设置端口转发 (443→8443)"（需管理员）
5. **启动服务**：点击"启动服务"

### 一键配置（推荐）

点击"一键配置并启动"按钮，自动完成所有配置步骤。

### 端口转发管理

**为什么需要端口转发？**
- Trae 访问 `https://api.openai.com:443`
- 本地服务监听 `127.0.0.1:8443`
- 需要将 443 端口转发到 8443

**操作步骤**：
1. 以管理员权限启动程序
2. 点击"设置端口转发 (443→8443)"
3. 点击"查看转发规则"确认设置成功

**取消转发**：
- 点击"取消端口转发"即可移除规则

### 检查更新

1. **配置更新服务器**（首次使用）：
   - 编辑 `.env` 文件
   - 设置 `UPDATE_CHECK_URL`

2. **检查更新**：
   - 菜单栏 → 帮助 → 检查更新
   - 查看版本对比和更新日志

### config.json 配置项

```json
{
  "listen_host": "127.0.0.1",         // 监听地址
  "listen_port": 8443,                // 监听端口
  "default_upstream": "https://newapi.loserrc.com",
  "verify_upstream_ssl": true,        // 验证上游 SSL（已修复）
  "intercept_hosts": [                // 劫持的域名
    "api.openai.com",
    "api.anthropic.com",
    "generativelanguage.googleapis.com"
  ]
}
```

## 📦 分发建议

### 方式 1：直接分发 EXE

**优点**：
- 单文件，方便分发
- 用户无需解压

**缺点**：
- 首次启动稍慢（解压到临时目录）
- 无法直接修改 .env（需在 EXE 同目录创建）

**适用场景**：
- 快速分发
- 不需要频繁修改配置

### 方式 2：分发文件夹

**优点**：
- 启动快速
- 可直接修改 `_internal\.env`

**缺点**：
- 文件较多
- 需要打包成 ZIP

**适用场景**：
- 需要自定义配置
- 企业内部部署

### 方式 3：制作安装包（推荐）

使用 Inno Setup 制作安装包：

```bash
# 已有配置文件
iscc install.iss

# 生成安装包
Output\Trae-Poxy-Setup.exe
```

**优点**：
- 专业的安装体验
- 自动创建快捷方式
- 支持卸载

## 🔒 安全建议
